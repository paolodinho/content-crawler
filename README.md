# Content Crawler Tool

A web-based tool to crawl and extract article content formatted for WordPress.

## Features

- Extract main content from any article URL
- Preserve heading structure (H1, H2, H3)
- Keep images
- Clean HTML output ready for WordPress
- One-click copy for Title, Content, or Both

## Usage

1. Enter an article URL
2. Click "Crawl Content"
3. Copy the formatted content to WordPress

## Deployment

This app is deployed on Railway.

## Tech Stack

- Backend: Flask (Python)
- Content Extraction: Trafilatura
- Frontend: HTML/CSS/JavaScript
